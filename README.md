# Blinkit-Sales-Analysis

🛒 Blinkit Sales Analysis Dashboard
📌 Overview

This is a Power BI project based on Blinkit sales data.
The dashboard shows sales performance, product analysis, outlet performance, and customer insights using interactive charts and KPIs.

🚀 Features
Total Sales Analysis
Product Category Analysis
Outlet Sales Performance
Monthly Sales Trends
Customer Rating Analysis
Interactive Filters & Charts
🛠 Tools Used
Microsoft Power BI
Microsoft Excel>
Power Query
DAX
📊 Insights
Supermarket Type 1 has highest sales.
Regular fat products sold more.
Tier 3 locations performed better.
Fruits and Snacks are top-selling categories.
